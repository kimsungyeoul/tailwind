# �� 벤더 프리픽스(Vendor Prefix)란 무엇인가?

**벤더 프리픽스(Vendor Prefix)** 란, 브라우저 엔진 제조사가 **자사 브라우저에만 실험적으로 도입한 CSS 기능을 제공**하기 위해 **CSS 속성 앞에 붙이는 접두사(prefix)**입니다.

쉽게 말해:

> “아직 표준으로 완전히 확정되지 않은 기능이지만, 우리 브라우저에서는 먼저 써볼 수 있게 해줄게요!”

라는 의미입니다.

---

# �� 왜 벤더 프리픽스가 존재할까?

CSS는 W3C 표준화 절차를 거치는데, **표준이 확정되기까지 시간(몇 년)**이 걸립니다.

따라서 브라우저 제조사들은:

1. 새로운 CSS 기능을 실험적으로 먼저 구현하고
2. 사용자(개발자)의 피드백을 받고
3. 안정화되면 표준으로 확정

하는 방식으로 발전시켜 왔습니다.

이때, **표준이 아니라 실험 기능임을 명확히 하기 위해** 벤더 프리픽스를 붙였습니다.

---

# �� 대표적인 벤더 프리픽스 목록

| 브라우저                    | 프리픽스       | 엔진             |
| ----------------------- | ---------- | -------------- |
| Chrome, Edge, Safari    | `-webkit-` | WebKit / Blink |
| Firefox                 | `-moz-`    | Gecko          |
| Internet Explorer(구 버전) | `-ms-`     | Trident        |
| Opera(Presto 엔진 시절)     | `-o-`      | Presto         |
